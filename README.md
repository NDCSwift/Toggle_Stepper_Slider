# 🎚️ Toggle Stepper Slider

A SwiftUI example demonstrating Toggle, Stepper, and Slider controls for iOS apps.

## 📖 What this is
This project shows how to use Toggle, Stepper, and Slider — three of the most common SwiftUI input controls. It covers state binding, styling options, and practical use cases for each component. A go-to reference when building forms or settings screens in SwiftUI.

## ✅ Why you'd use it
- **Three controls in one project** — Toggle, Stepper, and Slider all demonstrated side by side
- **State binding explained** — see exactly how each control reads and writes to app state
- **Ready to copy into your app** — clean, minimal examples you can adapt immediately

## 🚀 Getting Started
1. Clone the repo
2. Open `Toggle_Stepper_Slider.xcodeproj` in Xcode
3. Select your target device or simulator
4. Update the bundle ID if needed
5. Build and run

## 📝 Notes
- Built with SwiftUI
- Requires Xcode 14+

## ⚙️ Requirements
